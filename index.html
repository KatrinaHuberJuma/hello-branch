Git Branch Instructions

  1. git clone [[repo link]]
  2. git checkout -b [[branch name]] (this creates and checks out a branch)
  3. make edits and save file
  4. git commit -am "your commit message"
  5. git pull origin master
  6. if any conflicts:
    • open file in code editor to see conflicts
    • delete merge markers and correct any edits, save file
    • git commit -am "your commit message"
  7. git push origin [[branch name]]
  8. go into GitHub and make pull request
  9. someone else should approve the request
  10. after your request is approved, git pull origin master
  11. resume editing (step 3)
